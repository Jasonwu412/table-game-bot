## Useful commands

以下是一些常用的 Flutter / iOS 模拟器 命令，方便在本地跑起并调试应用：

- 启动 macOS 上的 iOS Simulator：

```bash
open -a Simulator
```

- 列出可用设备（包括模拟器和真机）：

```bash
flutter devices
```

- 在默认或已选择的设备上运行（若有多个设备会提示选择）：

```bash
flutter run
```

- 在指定设备上运行（替换为你的设备 ID）：

```bash
flutter run -d <你的设备ID>
```

- 以 release 模式构建并在设备上运行（更接近商用版本，关闭调试功能）：

```bash
flutter run --release
```

说明：如果需要停止正在运行的 `flutter run`，在终端按 `Ctrl+C`。
